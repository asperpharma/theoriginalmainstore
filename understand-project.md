## 13. Security Knowledge & Context
*   **Project Context:** Asper Beauty Shop is a production-bound, public-facing e-commerce application handling real customer data and transactions.
*   **Data Sensitivity:** The application processes potentially sensitive customer data, including health/skin concerns (medical history context), order details, and personal contact information. 
*   **User Context:** The storefront is accessible to public users, while the admin/backend (Supabase/Shopify) is strictly for trusted, authenticated staff and Edge Functions.
*   **Environment Context:** This repository represents the production-ready architecture. Any future development should assume a high-security posture, especially concerning Supabase Row Level Security (RLS) and the exposure of Shopify/Gemini API keys.
